# Nest + Prisma + MongoDB

## Description

- nestjs와 prisma를 사용하여 MongoDB를 사용하는 걸 연습하는 백엔드 프로젝트입니다.

## features

- User CRUD (OAtuh(discord, google, kakao) + JWT)
- Post CRUD (게시글 작성, 수정, 삭제, 조회, 좋아요, 댓글)
- Comment CRUD (댓글 작성, 수정, 삭제)

## Tech Stack

- NestJS
- Prisma
- MongoDB
- OAuth
- JWT

## Getting Started

```bash
$ pnpm install 
$ pnpm run start:dev
```
